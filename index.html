Helpful tips

Submit Early! The closer to the due date, the slower auto-grader gets :(

Linters suck but they don't have to.

If you write your code with linters in mind, it will save you a lot of time. Here are some tips:

  1. Read the Setup Guide & download the extensions
    • I also like the extension Auto Rename Tag, which renames opening and closing tags at the same time!
  2. Learn a couple vscode keyboard shortcuts!
  3. Give yourself around 30 minutes to work on your auto-grader tasks sometimes you hit submit for the first time and theres a lot of red.
  4. You can submit as many times as you want to make sure everything passes, so do it often!

Follow what makes you curious.

A lot of the assignments in this class are opened ended and allow for some creative flair, so chase what interests you!
